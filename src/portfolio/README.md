## 作品说明模板

后续每个作品建议都保留一篇详情文档，结构保持一致：

- 项目背景：为什么做这个作品，解决什么问题。
- 功能展示：核心页面、交互流程、上线地址。
- 技术实现：技术栈、架构选择、关键难点。
- 复盘总结：做得好的地方、可以继续改进的地方。
